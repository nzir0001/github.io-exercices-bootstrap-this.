What is Bootstrap

Bootstrap.css is a CSS framework that organizes and manages the layout of a website. So the HTML part manages the content and structure of a web page and CSS takes care of the layout of the site. For this reason, the two structures must coexist to perform a particular action.

It also provides tools with styles already in place eg for buttons, typography for navigation interfaces and many more.

What are the advantages of using Bootstrap

-It makes it easy and quicker for setting up a web page presentation

-Bootstrap allows the CSS presentation to be similar regardless of the browser used

-It has a grid system that allows the positioning of HTML elements on a web page
